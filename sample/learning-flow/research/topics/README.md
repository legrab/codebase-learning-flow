# Research Topics

Create one canonical file per stable topic ID.

Use `learning-flow/templates/research-topic.md`.

A topic should contain:

- scope and value;
- current understanding;
- architectural or domain map;
- evidence and repository references;
- known constraints;
- runtime or data questions;
- uncertainty;
- related materials;
- next research direction;
- validation history.

Keep teaching structure in `materials/themes/`. Keep contributor-specific interaction in `contributors/`.
