# Routing and behavior checks

These fixtures are lightweight regression cases, not a benchmark or scoring system.

- `agentic-cases.yaml` checks workflow configuration, existing and missing root integration, custom-harness mapping, layer separation, defaults, approval restraint, and evidence language.
- `minimal-cases.yaml` checks the single learning-skill branches, initial agentic-setup mapping, and persistence restraint.
- `full-cases.yaml` checks primary learning-skill routing, compact bootstrap coverage, and anti-ceremony behavior.

Review them when changing skill descriptions, root integration, agentic settings, planning behavior, baseline research, or understanding-check policy.
